Sharing Galleries

PhotoHouse generates unique, hard-to-guess URLs for each gallery. You control whether clients need a PIN, a password, or can access the gallery freely. Each event can have multiple share links — useful when sharing different group subsets with different people.

Creating a Share Link

  1. Open your event and click "Share Event"

    The share modal opens, showing any existing links and a form to create a new one.

  2. Choose an access type

    Select PIN, Password, or No protection.

  3. Set an optional expiry date

    Leave blank for a permanent link, or pick a date after which the link stops working.

  4. Click Create

    Copy the URL and send it to your client.

Access Types

PIN Recommended

A 4-digit numeric PIN is the easiest option for clients. They see four input boxes and type the PIN on their phone without needing to remember a complex password. You can reveal the PIN later from the share modal if a client forgets it.

Password

A text password (minimum 4 characters). Good when you want a memorable phrase — e.g. smithwedding2024 — that you can include in a welcome email.

No protection

Anyone with the link can view the gallery. The URL is still a random 16-character hex string, so it's not guessable, but there's no credential layer.

⚠️
Use protection for private events

Unprotected links can be forwarded. For weddings or private family sessions, always use a PIN or password.

What Clients See

After entering their PIN or password, clients land on a branded gallery page showing:

  • Your studio logo, name, and brand colour in the hero header.
  • The event name and date.
  • Group filter pills (if you created groups).
  • A Select Photos button to enter selection mode.
  • A Download All (ZIP) button (Pro/Studio plans only).

PIN Entry Screen

Clients who navigate directly to a PIN-protected gallery see a clean entry screen with your studio branding before any photos are revealed.

Revoking Links

Open the Share modal and click Revoke next to a link. The URL immediately stops working. Existing browser sessions with a valid cookie will expire within 24 hours.

Multiple Share Links

You can create multiple links per event with different settings — for example, a PIN link for the wedding couple and a separate password link for the parents. Each link can show a different subset of groups using group visibility overrides.
