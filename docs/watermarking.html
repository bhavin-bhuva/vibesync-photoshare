Watermarking

PhotoHouse applies your studio watermark to every photo a client downloads — protecting your work without affecting what they see in the gallery preview. Watermarking is available on Pro and Studio plans.

How Watermarking Works

Watermarks are applied at download time using the Sharp image processing library on the server. The original photo stored in S3 is never modified — only the file sent to the client's browser carries the watermark.

This means you can change your watermark settings at any time and every future download will use the new settings, including for existing events.

ℹ️
Pro & Studio only

Watermarking is disabled for Free accounts. Gallery photos for Free plan events are downloaded without a watermark.

Configuring Your Watermark

Go to Profile → Watermark Settings to configure:

Logo watermark

If you've uploaded a studio logo (in Studio Branding), it's automatically used as the watermark graphic. The logo is resized to 8% of the image width and composited at your chosen position.

Text watermark fallback

When no logo is set (or if the logo fails to load), PhotoHouse falls back to an SVG text watermark using your studio name with a subtle drop-shadow filter.

Position

Choose where the watermark appears:

  • Bottom right — default, least obtrusive for landscape shots
  • Bottom left
  • Bottom center

Opacity

Set opacity between 10% and 80%. Lower values create a subtle translucent mark; higher values create a more obvious brand stamp. The live preview in the settings panel updates in real time as you drag the slider.

Live Preview

The preview panel in Watermark Settings shows a simulated photo with your logo or studio name at the selected position and opacity. This is a CSS approximation — the actual composited watermark is generated by Sharp on the server and may differ slightly depending on photo dimensions.

Enabling and Disabling

Toggle Add watermark to downloaded photos to enable or disable watermarking. When disabled, all downloads (individual photos and ZIPs) are served without modification.

💡
Workflow tip

Disable watermarking before downloading your final delivered files, then re-enable it for ongoing client gallery access.
